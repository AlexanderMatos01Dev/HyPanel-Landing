## ğŸš€ Quick Start

### Prerequisites

- Node.js 18+ and npm installed
- Basic knowledge of Astro and React

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/AlexanderMatos01Dev/HyPanel-Landing.git
   cd HyPanel-Landing
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   http://localhost:4321
   ```
